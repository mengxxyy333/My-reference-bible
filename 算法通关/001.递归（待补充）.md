# 递归（图片问题暂未处理）

## 何为递归？

​	递归是一种自己调用自己的行为。

​	具体一点说，就是当处理大问题时，发现需要同样的规则求出稍小问题的解，才能求出大问题的解，这样不断地向下传递，一直到某一个边界，或者说小问题已经不能再继续向下分解为止。

## 例题

​	给定一个n个元素的数组a，求出该数组的最大值。

​	（*这里给出递归解法，初步理解什么是递归*）

​	思路：大问题是整个数组的最大值，那么我们分解成小问题，是否可以将数组根据中点分割为两份，先求前面的最大值，再求后面的最大值呢？答案是可以的，因为前半部分的最大值和后半部分的最大值，二者较大的值即为整个数组的最大值。那么这两部分的最大值怎么求呢？一样的过程，划分、整合，返回结果。例如，大小为5的数组的求解如下：

​	我们发现，递归进行到左右边界相同时，就不能继续向下分解了，这个时候需要返回我们的结果了。

​	一开始，从下标0到下标4，找最大值，等同于找[0, 2]上的最大值和[3, 4]上的最大值，二者大者返回；下标[0, 2]上的最大值等同于找[0, 1]上的最大值和[2, 2]上的最大值，二者大者返回。而[2, 2]上的最大值不就是a[2]本身吗？这个时候直接返回就行了。假如给出数组a：[4, 1, 5, 3, 6]，返回值过程如下：

## code

```cpp
#include <iostream>
using namespace std;

int a[105];

// f函数的作用是求[l, r]上的最大值 
int f(int l, int r)
{
	// 如果左右边界相同，直接返回结果 
	if(l == r)
	{
		return a[l];
	}
  
	// 取中点 
	int mid = (l+r) / 2;
	
	// lmax记录左半部分的最大值 
	int lmax = f(l, mid);
	
	// rmax记录右半部分的最大值 
	int rmax = f(mid+1, r);
	
	// 将二者大的作为结果返回 
	return max(lmax, rmax);
}

int main()
{
	// 自行处理数据输入 
	int n;
	cin >> n;
	for(int i = 0; i < n; i++)
	{
		cin >> a[i];
	}
	
	// 最终结果保存在ans中 
	int ans = f(0, n-1);
	
	cout << ans;
	
	return 0;
}
```

​	这样我们就初步理解了递归的过程。

## 递归的核心本质

​	递归是如何做到的呢？

​	实际上是在每次调用子问题时，将目前正在执行的函数暂时保管起来，当前的区间边界、求得的结果、变量等都保存，在进入子问题的函数中，为什么要保存那么多的东西呢？因为子问题调用结束返回时，还要函数本身继续正常运行，听起来有点玄乎，实际上很好理解。上面的例子中，我们先计算了lmax，接下来在计算rmax时，进入该子问题之前，我们应该保持lmax的值不变，这样在rmax返回时我们才能确保答案的正确。

​	那么我们没有做多余的动作，比如开一个临时变量存储lmax的值等等，程序是怎么记住的呢？实际上当前函数是被压入系统栈当中，包括一些必须的变量，都要记录在系统栈中，这个动作我们有一个比较形象的词语称呼：保护现场。没错，必须要保护现场，才能保证答案的正确性。

## 递归与非递归

​	**递归一定可以改成非递归**，无非就是不用系统栈，用内存空间呗。

​	系统栈较为脆弱，通常递归深度足够深的时候，就会报错了，俗称爆栈，通常递归都要改为非递归。

### 	哪些递归没必要改成非递归呢？

​	即使数据再大，递归的深度也不会很深。例如：归并排序、快速排序、线段树、很多平衡树等。

​	例如线段树，每次分一半，即使数据范围在2^64个，根据完全二叉树的性质可知，最大深度也就在64层左右，还会有提前弹出的过程，深度只少不多，这样的递归就不必改为非递归。

## master公式

​	master公式是用来快速计算递归的时间复杂度的公式，递归的形式需满足：T(N) = a*T(N/b) + O(n^c)。

​	使用master公式的前提是递归分解**子问题的规模必须相同**！

​	公式内容如下：

​		如果log(b,a) < c，复杂度为O(n^c)；

​		如果log(b,a) > c，复杂度为O(n^log(b,a))；

​		如果log(b,a) == c，复杂度为O(n^c * logn)；

​	特殊地：形如T(N) = 2*T(N/2) + O(n*logn)的递归，时间复杂度为O(n*(logn)^2)；

## 只用递归实现栈的逆序

​	我们可以利用栈本身可以保存这一层状态的这一特性，来做栈的逆序。每层递归保存当前的值，如果栈空了，开始放回即可。

​	每次保存一个值，这个值是当前栈底元素，且是递归获取，所以push的过程实际上是按取出顺序倒序压入，从而实现逆序。

​	给出栈：s[1, 2, 3, 4, 5]，如下所示：

<img src="typora-user-images\image-20241116214916610.png" alt="image-20241116214916610" style="zoom: 50%;" />

​	每次需要获取栈底元素并移除，返回给逆序函数，所以返回给逆序函数的顺序依次为：1, 2, 3, 4, 5。

​	而逆序函数也是递归调用的，所以压入栈的顺序其实应该为5, 4, 3, 2, 1。故而实现逆序。时间复杂度O(n^2)。

​	下面看看两个函数如何实现：

​	给定一个栈，获得的元素利用递归本身特性保存下来，待到最后一层往回压入，代码如下：

```cpp
void reversestack(stack<int>& s) {
	if (s.empty()) {
		return ;
	}
	int num = bottomout(s);
	reversestack(s);
	s.push(num);
}
```

​	获取栈底元素并删除的函数：从栈顶开始，每次保存当前值并向下沉直到栈底，返回的过程中除了栈底元素都重新压入即可。代码如下：

```cpp
// 返回栈底元素，并移除，上面的元素依次下沉
int bottomout(stack<int>& s) {
	int ans = s.top();
	s.pop();
	if (s.empty()) {
		return ans;
	}
	int last = bottomout(s);
	s.push(ans);
	return last;
}
```

## 只用递归实现栈的排序

​	如何只用递归把一个栈排序？需要实现一下几个方法，或者说步骤：1）递归到底，统计一共有几层，即栈的深度，假设为 h ；2）只需要往下递归 h 层，找出最大值，假设为 maxv；3）只需要往下递归 h 层，最大值是maxv，返回最大值出现了几次，假设为 cnt 次；4）将 cnt 个最大值沉到最下面的 [h - cnt + 1, h] 层，其余元素相对位置保持不变；5）对剩下的 h - cnt 个元素重复该操作，即令 h = h - cnt。

​	大体流程如下：

```c++
void fsort(stack<int>& stk) {
    int d = deep(stk); // 返回栈的深度
    while(d > 0) {
        int maxv = query_max(stk, d); // 向下递归d层，求出最大值
        int cnt = query_cnt(stk, d, maxv); // 向下递归d层，最大值是maxv，求出maxv出现了几次
        down(stk, d, maxv, cnt); // 将cnt个最大值沉底，其余元素相对位置不变
        d -= cnt;
    }
}
```

​	下面看四个方法具体如何实现：

​	求一个栈的深度，对于语言现有的库函数当然可以求，但是那样便显得无趣，我们说：只用递归来实现，在给定的栈中操作就可以了。我们利用递归保存现场的特性，每来到一层就保留当前层的值，然后弹出，去求剩余层的深度，剩余层的深度+1就是当前栈的深度，遇到栈为空就返回0即可，返回时依次再将元素压入栈中即可，相对位置没有发生变化。代码如下：

```c++
// 返回栈stk的深度
int deep(stack<int>& stk) {
    if(stk.empty()) {
        return 0;
    }
    int num = stk.top();
    stk.pop();
    int d = deep(stk) + 1;
    stk.push(num);
    return d;
}
```

​	求栈中元素的最大值，与上面方法的思路就类似了，记录当前num，弹出num，去剩下的栈中元素找出最大值，返回时与num决出最大值即可。代码如下：

```c++
// 给定一个栈，只用向下扎d层，返回这d层的最大值
int query_max(stack<int>& stk, int d) {
    if(d == 0) {
        return INT_MIN;
    }
    int num = stk.top();
    stk.pop();
    int restmax = query_max(stk, d - 1);
    int maxv = max(num, restmax);
    stk.push(num);
    return maxv;
}
```

​	求最大值出现的次数，思路同上，返回过程中看当前值是否等于最大值，等于就把答案+1继续向上返回即可。代码如下：

```c++
// 给定一个栈，只用向下扎d层，且这d层的最大值是maxv，返回maxv出现的次数
int query_cnt(stack<int>& stk, int d, int maxv) {
    if(d == 0) {
        return 0;
    }
    int num = stk.top();
    stk.pop();
    int restcnt = query_cnt(stk, d - 1, maxv);
    int cnt = restcnt + (num == maxv ? 1 : 0);
    stk.push(num);
    return cnt;
}
```

​	最后这个最大值沉底的方法，看起来较难，实际上思路也同上。我们在逐渐记录并弹出元素直至栈为空时，先压入cnt个最大值，在返回时遇到其他值再压入即可，如果是最大值就跳过不用管即可。代码如下：

```c++
// 给定一个栈，向下扎d层，最大值是maxv，且maxv出现了cnt次
// 函数的作用是：将cnt个maxv沉到当前最下面的层，其余元素相对位置保持不变
void down(stack<int>& stk, int d, int maxv, int cnt) {
    if(d == 0) {
        for(int i = 0; i < cnt; i++) {
            stk.push(maxv);
        }
    }
    else {
        int num = stk.top();
        stk.pop();
        down(stk, d - 1, maxv, cnt);
        if(num != maxv) {
            stk.push(num);
        }
    }
}
```

​	时间复杂度O(n^2)。
