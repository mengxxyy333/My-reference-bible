# 位图

***前置知识：二进制与位运算***

## 位图的原理

​	有一个集合，记录某个数字是否在这个集合里。听起来跟哈希表的作用是一样的，没错，哈希表随用随存随取，可以存任何没有溢出的数字，但是相比于位图，哈希表会更加占用空间，假设哈希表中元素类型是int，那么每存一个数字就花费32位，位图就是用位信息来标记某个数字是否出现过，那么位图有没有缺点呢？有的，位图存储的数字不是随意的，他只能存一段连续且确定的数字，例如：[0, 1000]范围内的数字的存储、取出、判断是否出现过。这样用一位来记录某个数字是否出现过，就节省了空间。位图的大小需要在初始化时指定，中途不能修改。

## 位图的实现

​	那么具体如何实现呢？假设想要判断[0, 100]范围的数字，那么只需要4个int整数共128位，就能放下101个数字。也就是说，我们想查找一个数字是否在位图中，只需要除法确定在哪个数字上，取余判断在该数字的哪位上即可。整体代码如下：

```c++
class bitSet {
	int n; // 位图大小：即能存储多少个连续的数字
	vector<int> bithash; // 具体存储的位图

public:
	bitSet(int size) {
		n = size;
		bithash.resize((n + 31) / 32, 0);
	}

	// 将num加入位图
	void add(int num) {
		bithash[num / 32] |= 1 << (num % 32);
	}

	// 将num从位图中移除
	void dele(int num) {
		bithash[num / 32] &= ~(1 << num % 32);
	}

	// 将num的信息反置，即：如果存在num，移除；如果不存在num，加入
	void rev(int num) {
		bithash[num / 32] ^= 1 << (num % 32);
	}

	// 判断num是否在位图中
	bool contains(int num) {
		return (bithash[num / 32] >> (num % 32)) & 1;
	}
};

```

```c++
class Bitset {
    int n;
    vector<int> bithash;
    int zeros;
    int ones;
    bool reversed;
public:
    Bitset(int size) {
        n = size;
        bithash.resize((size + 31) / 32, 0);
        zeros = size;
        ones = 0;
        reversed = false;
    }
    
    void fix(int idx) {
        if(!reversed) {
            if((bithash[idx / 32] & (1 << (idx % 32))) == 0) {
                zeros--;
                ones++;
                bithash[idx / 32] |= (1 << (idx % 32));
            }
        }
        else {
            if((bithash[idx / 32] & (1 << (idx % 32))) != 0) {
                zeros--;
                ones++;
                bithash[idx / 32] ^= (1 << (idx % 32));
            }
        }
    }
    
    void unfix(int idx) {
        if(!reversed) {
            if((bithash[idx / 32] & (1 << (idx % 32))) != 0) {
                zeros++;
                ones--;
                bithash[idx / 32] ^= (1 << (idx % 32));
            }
        }
        else {
            if((bithash[idx / 32] & (1 << (idx % 32))) == 0) {
                zeros++;
                ones--;
                bithash[idx / 32] |= (1 << (idx % 32));
            }
        }
    }
    
    void flip() {
        reversed = !reversed;
        int tmp = zeros;
        zeros = ones;
        ones = tmp;
    }
    
    bool all() {
        return ones == n;
    }
    
    bool one() {
        return ones > 0;
    }
    
    int count() {
        return ones;
    }
    
    string toString() {
        string ans = "";
        for(int i = 0, k = 0, num, status; i < n; k++) {
            num = bithash[k];
            for(int j = 0; j < 32 && i < n; j++, i++) {
                status = (num >> j) & 1;
                status ^= reversed ? 1 : 0;
                ans += status + '0';
            }
        }
        return ans;
    }
};
```
