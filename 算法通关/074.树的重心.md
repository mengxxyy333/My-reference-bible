# 树的重心

​	***前置知识：链式前向星、树型dp、树上倍增和LCA***

## 树的重心的定义

​	有三种定义，但是都是同一个结果：

​		1、以某个节点为根时，最大子树的节点数最少，那么这个节点是重心。

​		2、以某个节点为根时，每棵子树的节点数不超过总节点数的一半，那么这个节点是重心。

​		3、以某个节点为根时，所有节点走向该节点的总边数最少，那么这个节点是重心。

## 树的重心的性质

​	1、一棵树上最多有两个重心，如果有两个重心，那么两个重心一定相邻。

​	2、如果树上增加或者删除一个叶节点，那么转移后的重心最多移动一条边，或者不动。

​	3、如果把两棵树连起来，那么新树的重心一定在原来两棵树重心的路径上。

​	4、树上的边权如果都为正数，不管边权如何分布，，所有节点都走向重心的总距离和最小。

## 代码实现（模板题）

​	题目链接：https://www.luogu.com.cn/problem/U104609、http://poj.org/problem?id=1655

​	两道题目在最后答案收集时的细节不同，其余均相同。这里给出的是洛谷的代码。poj 需要自行注册账号，且编译器较老，可能需要调试，有时间精力的小伙伴自行修改代码并尝试提交。

### 定义一

​	先来按照定义一找到树的重心。当来到一个节点时，不能傻乎乎地去遍历求解每棵子树的大小，需要快速知道该信息。设置一个 sz 数组，sz[u] 表示节点 u 及其所有子树的节点个数，假设 u 的子节点分别为 a、b、c，那么 u 的这三个子树的大小可以获取：sz[a]、sz[b]、sz[c]，假设 u 的父节点是 f，那么以 u 为根的情况下，f 也是 u 的子树，那么如何求解 sz[f] 呢？一棵树，除了 u 下方的节点就是 f 及其上方的节点，所以只需要 n - sz[u] 就能得到 sz[f] 了。这个过程直接 DFS 就可以了，最后加工答案，可以看下方AC代码：

```c++
#include <bits/stdc++.h>
using namespace std;

#define MAXN 50005
const int mod = 1e9 + 7;
typedef long long ll;

int n;
int sz[MAXN];
int best = INT_MAX, center;
vector<int> ans;

int head[MAXN];
int to[MAXN << 1];
int nxt[MAXN << 1];
int cnt  = 1;
void addedge(int u, int v) {
	nxt[cnt] = head[u];
	to[cnt] = v;
	head[u] = cnt++;
}

void dfs(int u, int fa) {
	sz[u] = 1;
	int tmp = 0; // 记录节点u的各个子树的最大节点数
	for(int e = head[u]; e != 0; e = nxt[e]) {
		if(to[e] != fa) {
			dfs(to[e], u);
			sz[u] += sz[to[e]];
			tmp = max(tmp, sz[to[e]]);
		}
	}
	tmp = max(tmp, n - sz[u]);
	if(tmp < best) {
		best = tmp;
		ans.clear();
		ans.push_back(u);
	}
	else if(tmp == best) {
        // 题目要求从小到大输出
		if(u >= ans[0]) ans.push_back(u);
		else {
			ans.push_back(u);
			swap(ans[0], ans[1]);
		}
	}
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	
	cin >> n;
	for(int i = 1; i <= n - 1; i++) {
		int u, v; cin >> u >> v;
		addedge(u, v);
		addedge(v, u);
	}
	dfs(1, 0);
	for(int x : ans) cout << x << " ";

	return 0;
}
```

### 定义二

​	再来按照定义二来找到树的重心。定义一和二是解题过程中常遇到的，三作为扩展内容。整体思路跟上方的差不多，求出每棵子树的大小，与 n / 2 对比就好了。或者使用一个 maxsub 数组，记录每个节点的子树中最大节点数，然后再看哪些节点是符合条件的，均可。下方代码实现的是前者：

```c++
#include <bits/stdc++.h>
using namespace std;

#define MAXN 50005
#define MAXM 100005
const int mod = 1e9 + 7;
typedef long long ll;

int n;
int sz[MAXN];
vector<int> ans;

int head[MAXN];
int to[MAXN << 1];
int nxt[MAXN << 1];
int cnt  = 1;
void addedge(int u, int v) {
	nxt[cnt] = head[u];
	to[cnt] = v;
	head[u] = cnt++;
}

void dfs(int u, int fa) {
	sz[u] = 1;
	bool f = false;
	for(int e = head[u]; e != 0; e = nxt[e]) {
		if(to[e] != fa) {
			dfs(to[e], u);
			sz[u] += sz[to[e]];
			if(sz[to[e]] > n / 2) {
				f = true;
				break;
			}
		}
	}
	if(n - sz[u] > n / 2) f = true;
	if(!f) ans.push_back(u);
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	
	cin >> n;
	for(int i = 1; i <= n - 1; i++) {
		int u, v; cin >> u >> v;
		addedge(u, v);
		addedge(v, u);
	}

	dfs(1, 0);

	if(ans.size() == 1) cout << ans[0];
	else {
		if(ans[0] > ans[1]) cout << ans[1] << " " << ans[0];
		else cout << ans[0] << " " << ans[1];
	}

	return 0;
}
```

## 有点权的树求重心以及最短路

​	题目链接：https://www.luogu.com.cn/problem/P2986

​	这道题目给出的树带边权以及点权，这是与模板题不同的地方。根据性质4，只要边权为正，那么所有点走向重心距离和最小，所以边权在求解树的重心过程中是没用的。那么点权呢？点权会影响树的重心，我们令 size[u] 表示以 u 为根的这棵子树的各节点权值之和。可以发现：模板题目中，所有点的点权为1。那么求解过程其实与模板题目很像了，只不过在求解节点 u 上方的子树总点权时，需要用点权总和减去 size[u]，而不是模板题目中的 n 了。

​	找到了树的重心之后，如何求解所有节点到该重心节点的路径和呢？只需要从重心节点跑一次 dfs 即可。具体细节见AC代码：

```c++
#include <bits/stdc++.h>
using namespace std;

#define MAXN 100005
#define MAXM 100005
const int mod = 1e9 + 7;
typedef long long ll;

int n;
int cows[MAXN];
int cowsum; // 记录总点权
int best = INT_MAX, center;
ll ans;

int head[MAXN];
int to[MAXN << 1];
int nxt[MAXN << 1];
int weight[MAXN << 1];
int cnt = 1;
void addedge(int u, int v, int w) {
	nxt[cnt] = head[u];
	to[cnt] = v;
	weight[cnt] = w;
	head[u] = cnt++;
}

int sz[MAXN]; // sz[i]表示：以i为根的子树总的点权和

// 找到树的重心
void findCenter(int u, int fa) {
	sz[u] = cows[u];
	int maxsub = 0;
	for(int e = head[u]; e != 0; e = nxt[e]) {
		if(to[e] != fa) {
			findCenter(to[e], u);
			sz[u] += sz[to[e]];
			maxsub = max(maxsub, sz[to[e]]);
		}
	}
	maxsub = max(maxsub, cowsum - sz[u]);
	if(maxsub < best) {
		best = maxsub;
		center = u;
	}
}

// 求解重心节点到所有节点的路径和
// pathsum表示：当前走到节点u，路径有多长
void dfs(int u, int fa, ll pathsum) {
	ans += (ll) cows[u] * pathsum;
	for(int e = head[u]; e != 0; e = nxt[e]) {
		if(to[e] != fa) {
			dfs(to[e], u, pathsum + weight[e]);
		}
	}
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	
	cin >> n;
	for(int i = 1; i <= n; i++) {
		cin >> cows[i];
		cowsum += cows[i];
	}
	for(int i = 1; i <= n - 1; i++) {
		int u, v, w; cin >> u >> v >> w;
		addedge(u, v, w);
		addedge(v, u, w);
	}
	findCenter(1, 0);
	dfs(center, 0, 0l);
	cout << ans;

	return 0;
}
```

## 增删边使其重心唯一

​	题目链接：https://codeforces.com/problemset/problem/1406/C

​	这道题目也很简单，我们只需要先确定重心的数量，如果是1，那就不用删增，但是这道题目要求必须删增，不然不能通过，所以就把那个唯一的重心连接的任意一个边删除再加上就好了。如果确实两个重心，假设是 c1 和 c2，为什么会有两个重心？单看一个重心例如 c1，一定是 c1 一侧的子树节点数比 c2 这棵子树的节点数少一个，反之亦然。那么我们只需要找到 c2 这棵子树的任意一个叶结点，将其与父节点这条边删除，再将这个叶节点直接挂在 c1 下方，重心就唯一了。详细实现见AC代码：

```c++
#include <bits/stdc++.h>
using namespace std;

#define MAXN 100005
#define MAXM 100005
const int mod = 1e9 + 7;
typedef long long ll;

int n;
int maxsub[MAXN];
int sz[MAXN];
int centers[2];
int leaf, leafFather;

int head[MAXN];
int to[MAXN << 1];
int nxt[MAXN << 1];
int cnt;
void addedge(int u, int v) {
	nxt[cnt] = head[u];
	to[cnt] = v;
	head[u] = cnt++;
}

void build() {
	cnt = 1;
	memset(head, 0, sizeof(head));
}

void dfs(int u, int fa) {
	sz[u] = 1;
	maxsub[u] = 0;
	for(int e = head[u]; e != 0; e = nxt[e]) {
		if(to[e] != fa) {
			dfs(to[e], u);
			sz[u] += sz[to[e]];
			maxsub[u] = max(maxsub[u], sz[to[e]]);
		}
	}
	maxsub[u] = max(maxsub[u], n - sz[u]);
}

// 找到重心并收集，返回重心数量
int centerCnt() {
	int m = 0;
	for(int i = 1; i <= n; i++) {
		if(maxsub[i] <= n / 2) centers[m++] = i;
	}
	return m;
}

// 一路dfs找到叶结点，记录一个叶结点与其父节点编号就返回
void findLeaf(int u, int fa) {
	for(int e = head[u]; e != 0; e = nxt[e]) {
		if(to[e] != fa) {
			findLeaf(to[e], u);
			return ;
		}
	}
	leaf = u;
	leafFather = fa;
}

void solve() {
	cin >> n;
	build();
	for(int i = 1; i <= n - 1; i++) {
		int u, v; cin >> u >> v;
		addedge(u, v);
		addedge(v, u);
	}
	dfs(1, 0);
	if(centerCnt() == 1) {
		cout << centers[0] << " " << to[head[centers[0]]] << "\n";
		cout << centers[0] << " " << to[head[centers[0]]] << "\n";
	}
	else {
		findLeaf(centers[1], centers[0]);
		cout << leafFather << " " << leaf << "\n";
		cout << centers[0] << " " << leaf << "\n";
	}
}

int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	
	int t; cin >> t;
	while(t--) {
		solve();
	}

	return 0;
}
```
