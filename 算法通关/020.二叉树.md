# 二叉树

***前置知识：递归***

​	二叉树是一种特殊的树，每个节点最多有两个孩子，没有孩子的部分指向空。二叉树规律明显，尤其是完全二叉树，可以用数组来表示，何为完全二叉树下面会介绍。在新手入门递归时，难免会从二叉树开始，因为树的结构本身就是递归的定义。二叉树的前中后序遍历都是递归的形式，代码简短且优雅，这也是递归一直以来的魅力。下面走进二叉树的世界。

## 二叉树的代码实现

​	数组对应关系方便竞赛环境，这里采用指针方式，也就是左右孩子表示法，但是存储还是需要数组来存储的。

```c++
#include <iostream>
using namespace std;

//树中的每个结点
template<typename T>
struct Treenode {
	T data;
	Treenode* left;
	Treenode* right;

	Treenode() : data(0), left(nullptr), right(nullptr) {}
	Treenode(T data) : data(data), left(nullptr), right(nullptr) {}
};

//二叉树类
template<typename T>
class BiTree {
private:
	Treenode<T>* nodes;
	Treenode<T>* root;
	int maxsize;

	void visit(Treenode<T>* node);
	Treenode<T>* create(T arr[], int size, int nodeid, T nullnode);
	void preOrder(Treenode<T>* node);
	void inOrder(Treenode<T>* node);
	void postOrder(Treenode<T>* node);
public:
	BiTree() : nodes(new Treenode<T>* [10005]), root(nullptr), maxsize(10005) {}
	BiTree(int maxsize) : nodes(new Treenode<T>[maxsize]), root(nullptr), maxsize(maxsize) {}
	Treenode<T>* GetTreenode(int id);
	void createtree(T arr[], int size, T nullnode);
	void preOrderTraversal();
	void inOrderTraversal();
	void postOrderTraversal();
	~BiTree();
};

//获取结点的值
template<typename T>
void BiTree<T>::visit(Treenode<T>* node) {
	cout << node->data;
}

//获取结点
template<typename T>
Treenode<T>* BiTree<T>::GetTreenode(int id) {
	return &nodes[id];
}

//创建结点
template<typename T>
Treenode<T>* BiTree<T>::create(T arr[], int size, int nodeid, T nullnode) {
	if (nodeid >= size || arr[nodeid] == nullnode) {
		return NULL;
	}

	Treenode<T>* newnode = GetTreenode(nodeid);
	newnode->data = arr[nodeid];
	newnode->left = create(arr, size, nodeid * 2, nullnode);
	newnode->right = create(arr, size, nodeid * 2 + 1, nullnode);

	return newnode;
}

//创建树
template<typename T>
void BiTree<T>::createtree(T arr[], int size, T nullnode) {
	root = create(arr, size, 1, nullnode);
}

//前序遍历的实现
template<typename T>
void BiTree<T>::preOrder(Treenode<T>* node) {
	if (node) {
		cout << node->data << " ";
		preOrder(node->left);
		preOrder(node->right);
	}
}

//前序遍历
template<typename T>
void BiTree<T>::preOrderTraversal() {
	preOrder(root);
}

//中序遍历的实现
template<typename T>
void BiTree<T>::inOrder(Treenode<T>* node) {
	if (node) {
		inOrder(node->left);
		cout << node->data << " ";
		inOrder(node->right);
	}
}

//中序遍历
template<typename T>
void BiTree<T>::inOrderTraversal() {
	inOrder(root);
}

//后序遍历的实现
template<typename T>
void BiTree<T>::postOrder(Treenode<T>* node) {
	if (node) {
		postOrder(node->left);
		postOrder(node->right);
		cout << node->data << " ";
	}
}

//后序遍历
template<typename T>
void BiTree<T>::postOrderTraversal() {
	postOrder(root);
}

//销毁这棵树
template<typename T>
BiTree<T>::~BiTree() {
	delete[] nodes;
}

int main() {
	const char nullnode = '-';
	char arr[15] = {
		nullnode,'a','b','c','d',
		nullnode,'e','f','g','h',
		nullnode,nullnode,nullnode,nullnode,'i'
	};

	BiTree<char> T(15);
	T.createtree(arr, 15, nullnode);
	T.preOrderTraversal();
	cout << endl;
	T.inOrderTraversal();
	cout << endl;
	T.postOrderTraversal();
	cout << endl;

	return 0;
}
```

### 非递归实现

#### 先序遍历

​	注意压栈的时候先压右孩子，再压左孩子，因为弹出的时候会逆序，即先弹出左孩子，再弹出右孩子。

​	AC代码：

```c++
vector<int> preorderTraversal(TreeNode* root) {
    vector<int> ans;
    if(root != nullptr) {
        stack<TreeNode*> stk;
        stk.push(root);
        while(!stk.empty()) {
            TreeNode* node = stk.top(); stk.pop();
            ans.push_back(node->val);
            if(node->right != nullptr) stk.push(node->right);
            if(node->left != nullptr) stk.push(node->left);
        }
    }
    return ans;
}
```

#### 中序遍历

​	按照如下步骤进行：对于任意一棵子树，先将左侧所有节点压入栈中，然后弹出栈顶元素，即最左侧的元素，每次弹出就记录答案。再去栈顶元素的右子树，重复上述步骤。这样做就模拟了栈的中序遍历：对于左侧节点，肯定能撸多深就撸多深，处理完最左侧的节点后，如果站在整棵树的角度来看，就是处理完所有左子树，再处理自己（即弹出栈并记录答案），最后处理右子树，对于右子树也是同样的处理方式。

​	AC代码：

```c++
vector<int> inorderTraversal(TreeNode* root) {
    vector<int> ans;
    if(root != nullptr) {
        stack<TreeNode*> stk;
        while(!stk.empty() || root != nullptr) {
            if(root != nullptr) {
                stk.push(root);
                root = root->left;
            }
            else {
                root = stk.top(); stk.pop();
                ans.push_back(root->val);
                root = root->right;
            }
        }
    }
    return ans;
}
```

#### 后序遍历

​	分为两个版本，一个版本用两个栈，另外一个版本用一个栈。不用想，看一眼就知道一个栈的版本更好，两个栈的版本纯属借用先序遍历，好理解简单。两个栈的实现方式是这样的：后序遍历顺序是左右中，先序遍历稍微调换一下，顺序是中右左（习惯的是中左右），这样将先序的中右左顺序翻转，就是后续的左右中了，代码实现上调换一下左右孩子压栈的顺序，弹出节点时，不打印值，而是放入另一个栈作为翻转结果即可。	

​	另外一个版本只需要用一个栈，我们需要用一个节点来记录上一次访问过的节点，因为左右中这个顺序，我们一定先去处理左子树，再去处理右子树，最后访问该根节点，所以必须知道上次访问到哪里了。如果左子树不为空，且上一次访问的既不是左子树也不是右子树，那就将左子树压入栈中；如果左子树为空，或者上一次访问的是左子树，那就去判断右子树，如果右子树不为空，且上一次访问的不是右子树，就将右子树压入栈中；如果右子树为空，或者上一次访问的是右子树，那么就说明左右子树都处理完成，直接访问根节点记录答案即可。我们直接使用题目给出的根节点root作为标记上一次访问的节点，如果还一直没处理过，root就是根节点，也就是此时还没记录任何答案，一直在递归访问左子树过程中。具体实现读者自行用代码模拟一下就懂了。

​	两个栈的版本（好写但是不推荐，因为额外空间复杂度为O(n)）：

```c++
vector<int> postorderTraversal(TreeNode* root) {
    vector<int> ans;
    if(root != nullptr) {
        stack<TreeNode*> stk;
        stack<TreeNode*> rev;
        stk.push(root);
        while(!stk.empty()) {
            TreeNode* node = stk.top(); stk.pop();
            rev.push(node);
            if(node->left != nullptr) stk.push(node->left);
            if(node->right != nullptr) stk.push(node->right);
        }
        while(!rev.empty()) {
            ans.push_back(rev.top()->val);
            rev.pop();
        }
    }
     return ans;
}
```

​	一个栈的版本：

```c++
vector<int> postorderTraversal(TreeNode* root) {
    vector<int> ans;
    if(root != nullptr) {
        stack<TreeNode*> stk;
        stk.push(root);
        while(!stk.empty()) {
            TreeNode* node = stk.top();
            if(node->left != nullptr && root != node->left && root != node->right) stk.push(node->left);
            else if(node->right != nullptr && root != node->right) stk.push(node->right);
            else {
                ans.push_back(node->val);
                root = stk.top(); stk.pop();
            }
        }
    }
    return ans;
}
```

### 时间复杂度分析

​	对于递归版本，每个节点来到三次，有n个节点，时间复杂度就是O(n)。非递归方法，每个节点进栈出栈一次，可能不是1次，例如两个栈的版本，但是次数有限，时间复杂度也是O(n)。

​	空间复杂度为O(h)，h为二叉树的高度。

### 预告

​	有一种算法，时间复杂度为O(n)，空间复杂度为O(1)，优于上面的算法，这种算法是Morris遍历，属于较难内容，后面会讲到。

### 二叉树的序列化和反序列化

​	题目链接：https://leetcode.cn/problems/serialize-and-deserialize-binary-tree/

#### 先序序列化

​	二叉树的序列化是指，一个二叉树可以表示为一个字符串的形式，该表示形式是唯一的，即：通过反序列化能够得到唯一的与之对应的二叉树。这里就需要对空节点也进行处理，例如用 '#' 符号来表示。

​	二叉树的序列化有几种方式，可以先序序列化，但是不能中序序列化，因为不同二叉树中序序列化的结果可能相同，例如：1,1->right= 2，与2,2->left = 1。后序序列化也是可以的，因为和先序差不多，下面看力扣AC代码：

```c++
class Codec {
public:
    void f(TreeNode* root, string &ans) {
        if(root == nullptr) {
            ans += "#,";
        }
        else {
            ans += to_string(root->val);
            ans += ",";
            f(root->left, ans);
            f(root->right, ans);
        }
    }

    // Encodes a tree to a single string.
    string serialize(TreeNode* root) {
        string ans = "";
        f(root, ans);
        return ans;
    }

    int cnt;
    TreeNode* g(vector<string>& vals) {
        string val = vals[cnt++];
        if(val == "#") return nullptr;
        TreeNode* head = new TreeNode(stoi(val));
        head->left = g(vals);
        head->right = g(vals);
        return head;
    }

    // Decodes your encoded data to tree.
    TreeNode* deserialize(string data) {
        vector<string> t;
        string tmp = "";
        for(int i = 0; i < data.size(); i++) {
            if(data[i] != ',') tmp += data[i];
            else {
                t.push_back(tmp);
                tmp.clear();
            }
        }
        cnt = 0;
        return g(t);
    }
};
```

#### 层序序列化

​	一样的题目链接，与先序序列化不同的是遍历方式，层序序列化需要按层遍历，但是在此题中，我们就不需要关心层数了，也就是size不需要了，按个节点去序列化与反序列化即可，为什么呢？首先，按层遍历得到的序列肯定是一层一层的节点顺序，而反序列化时，使用一个变量idx来标记都使用过哪些节点即可保证正确。什么意思？好比一棵三层的满二叉树，编号为1,2,3,4,5,6,7，这里层序遍历也是这个顺序，序列化后的结果想要反序列化，每遇到一个数字（不同的节点），例如当前是1，那么后续去建立1的左右孩子节点就顺着往下走2和3。此时队列剩余2和3，弹出队头是2，先处理2，再处理2的左右孩子，idx继续往下走是4和5，同理3处理完成后idx继续往下走，3的孩子就是6和7，结论是正确的。

​	AC代码如下：

```c++
class Codec {
public:
    TreeNode* q[10005];
    int l, r;

    // Encodes a tree to a single string.
    string serialize(TreeNode* root) {
        string ans = "";
        if(root != nullptr) {
            ans += to_string(root->val) + ",";
            l = r = 0;
            q[r++] = root;
            while(l < r) {
                root = q[l++];
                if(root->left != nullptr) {
                    ans += to_string(root->left->val) + ",";
                    q[r++] = root->left;
                }
                else ans += "#,";
                if(root->right != nullptr) {
                    ans += to_string(root->right->val) + ",";
                    q[r++] = root->right;
                }
                else ans += "#,";
            }
        }
        return ans;
    }

    TreeNode* generate(string s) {
        return s == "#" ? nullptr : new TreeNode(stoi(s));
    }

    // Decodes your encoded data to tree.
    TreeNode* deserialize(string data) {
        if(data == "") return nullptr;
        vector<string> t;
        string tmp = "";
        for(int i = 0; i < data.size(); i++) {
            if(data[i] != ',') tmp += data[i];
            else {
                t.push_back(tmp);
                tmp.clear();
            }
        }
        int idx = 0;
        l = r = 0;
        TreeNode* root = generate(t[idx++]);
        q[r++] = root;
        while(l < r) {
            TreeNode* cur = q[l++];
            cur->left = generate(t[idx++]);
            cur->right = generate(t[idx++]);
            if(cur->left) q[r++] = cur->left;
            if(cur->right) q[r++] = cur->right;
        }
        return root;
    }
};
```

### 最近公共祖先

​	这是大名鼎鼎的LCA问题，本节讨论的都是求两个节点的最近公共祖先，以后会涉及到更难的批量查询、在线查询LCA的问题，会涉及到Tarjan算法以及树链剖分等算法。

​	什么是最近公共祖先？有两个节点p和q，他们的祖先节点中，最深的，也就是最靠近这两个节点的祖先节点。

​	一种特殊情况：p在q的子树上或者q在p的子树上，那么他们的最近公共祖先就是q或者p。其余情况就是pq分属于两颗子树，谁拥有这两棵子树，谁就是LCA。

​	普通二叉树上寻找两个节点的LCA：https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-tree/

​	AC代码如下：

```c++
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
    if(root == nullptr || root == p || root == q) return root;
    TreeNode* l = lowestCommonAncestor(root->left, p, q);
    TreeNode* r = lowestCommonAncestor(root->right, p, q);
    if(l != nullptr && r != nullptr) return root;
    if(l == nullptr && r == nullptr) return nullptr;
    return l == nullptr ? r : l;
}
```

​	该代码读者可以自行画出一棵二叉树，分别代入p和q在一棵子树以及不在一棵子树的情况，就能明白代码的逻辑了。确保您模拟过这个过程，再来看我如下的代码说明：一旦找到空节点或者p或者q，直接返回；l 不为空且 r 也不为空，说明在当前子树root的左右两棵子树上分别找到p和q，返回root这个节点就是答案，再往上返回右边一直都是空返回值，因为p和q都已经出现过了，根据最后一行，一个空一个不空，返回不空的节点，答案结点就被最终返回了，这就是pq分开的情况；pq在同一棵子树的情况，第一行代码给出，一旦遇到p或q，直接返回，最终将p或q中先遇到的那个返回，也是正确答案。

​	搜索二叉树上寻找两个节点的LCA：https://leetcode.cn/problems/lowest-common-ancestor-of-a-binary-search-tree/

​	这个题目就要好好利用二叉搜索树的性质了。我们考虑这样一种情况：当前来到节点的值比p和q的值都小，说明p和q在当前节点的右子树上，那么去右子树找；如果当前节点的值比p和q的值都大，说明p和q在当前节点的左子树上，那么去左子树找。好了，这种情况之外是什么样？就是当前节点的值处理p和q的值之间，此时说明p和q分属于当前节点的左右子树上，那么当前节点就是答案。有没有可能还有更下面的公共祖先了呢？假设如果有，那么该节点值处于p和q之间，而此时p和q一定全位于当前节点的左子树或者右子树上，不可能分属于左右两棵树，而这与当前节点值处于p和q之间矛盾，故第一个遇到的节点值处于p和q之间的节点就是p和q的LCA。还剩下最特殊的情况：当前节点值等于p或者q，此时先遇到的就是答案，因为不管先遇到哪个，另一个一定在该节点的子树上。三种情况讨论完毕，覆盖所有情况。

​	AC代码如下：

```c++
TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {
    while(root->val != p->val && root->val != q->val) { // 没遇到p和q就继续找，先遇到谁就返回谁
        if(min(p->val, q->val) < root->val && root->val < max(p->val, q->val)) break; // 遇到值处于p和q之间的节点，就是答案
        root = root->val < min(p->val, q->val) ? root->right : root->left; // 当前节点值比p和q最小值小，就去左子树找；反之去右子树
    }
    return root;
}
```

### 带路径的递归问题（回溯、恢复现场）

​	题目链接：https://leetcode.cn/problems/path-sum-ii/

​	思路很简单，从根节点依次向下递归，找到叶子结点，沿途收集和与各个节点的值，到达叶子结点后，看与目标和是否一致，一致就收集答案，否则不收集。当从叶子结点返回时，需要将当前节点从答案中拿出去，这就是回溯，或者说恢复现场，例如调用左边失败后，需要将左孩子拿出去，再去右侧递归放入右孩子，不可能左右孩子都放，那就不行了。AC代码如下：

```c++
void f(TreeNode* root, int targetSum, int curSum, vector<vector<int>>& ans, vector<int>& path) {
    curSum += root->val;
    path.push_back(root->val);
    if(root->left == nullptr && root->right == nullptr) {
        if(curSum == targetSum) ans.push_back(path);
        else {
            path.erase(path.end() - 1);
            return;
        }
    }
    if(root->left != nullptr) f(root->left, targetSum, curSum, ans, path);
    if(root->right != nullptr) f(root->right, targetSum, curSum, ans, path);
    path.erase(path.end() - 1);
}
vector<vector<int>> pathSum(TreeNode* root, int targetSum) {
    vector<vector<int>> ans;
    if(root == nullptr) return ans;
    vector<int> path;
    f(root, targetSum, 0, ans, path);
    return ans;
}
```

### 二叉树打家劫舍（树形dp）

​	题目链接：https://leetcode.cn/problems/house-robber-iii/

​	打家劫舍系列问题可以说是dp入门很经典的题目，后续还有类似问题dp章节再讨论。

​	对于任意一棵子树X来说，要么就偷X的头节点，要么就不偷X的头节点。那么X如何结算能偷的最大价值呢？左子树偷一下，右子树偷一下，就行了。那么对于左子树和右子树，还是偷不偷头节点的问题，但是这时候就被X的头节点限制住了，也就是如果偷了X的头节点，那么左右子树的头节点一定不会被偷；如果没偷X的头节点，那么左右子树的头节点可偷可不偷，取最大值即可。

​	实现上，我们使用两个全局变量yes和no，分别表示跑完一棵子树后偷头节点可以获得的最大收益，以及不偷头节点可以获得的最大收益。对于空节点，二者均返回0，不必多说。对于普通节点，需要根据左右子树的结果来更新yes和no，对于左子树来说，当前头节点偷的情况下，左子树不能偷，所以累加全局的no变量；当前头节点不偷的情况下，左子树的头节点可偷可不偷，取max(yes, no)即可，注意：这里的yes和no是已经跑完左子树的递归了，所以代表的就是左子树偷头节点以及不偷所能带来的最大收益，右子树也同理，两边做和就是当前节点的最大收益，继续向上返回。最后yes和no记录的就是整棵树偷头节点以及不偷头节点所能带来的最大收益，返回最大值即可。AC代码如下：

```c++
class Solution {
public:
    int yes, no; // yes表示当前子树偷头节点可以获得的最大收益，no表示不偷头节点......
    int rob(TreeNode* root) {
        if(root == nullptr) {
            yes = 0;
            no = 0;
        }
        else {
            int y = root->val, n = 0; // 当前节点偷与不偷头节点带来的收益
            rob(root->left); // 去左子树偷
            y += no;
            n += max(yes, no);
            rob(root->right); // 去右子树偷
            y += no;
            n += max(yes, no);
            yes = y; // 此时y和n就是当前节点偷与不偷的最大收益
            no = n;
        }
        return max(yes, no);
    }
};
```
